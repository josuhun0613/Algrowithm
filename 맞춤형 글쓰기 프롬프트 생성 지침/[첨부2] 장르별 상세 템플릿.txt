
# 장르별 상세 템플릿

이 파일은 맞춤형 글쓰기 프롬프트 생성을 위한 장르별 세부 정보를 담고 있습니다.
아래 내용을 적용할때 인용코드를 쓰지마세요.

## 📖 1. 장르별 목차 구성 템플릿

### 수필 (에세이) 목차 구성
```

- 작가의 말 (3-5p)
- 1부: [주제 도입] (40-50p)
  ├ 1장. [개인적 경험으로 시작] (15-20p)
  ├ 2장. [문제의식 발견] (12-18p)
  └ 3장. [첫 번째 깨달음] (13-17p)
- 2부: [주제 심화] (50-70p)
  ├ 4장. [갈등과 혼란] (18-25p)
  ├ 5장. [타인과의 만남] (15-22p)
  └ 6장. [새로운 시각] (17-23p)
- 3부: [주제 승화] (40-60p)
  ├ 7장. [변화의 시작] (15-20p)
  ├ 8장. [실천과 적용] (12-18p)
  └ 9장. [현재의 나] (13-17p)
- 에필로그 (5-10p)

🎯 **각 장별 작성 컨셉**
- 1장: 독자의 관심을 끄는 강렬한 개인 경험
- 2장: 보편적 공감을 이끌어내는 문제 제기
- 3장: 첫 번째 작은 깨달음과 변화의 조짐
- 4장: 혼란과 갈등을 통한 깊이 있는 성찰
- 5장: 타인과의 관계에서 얻는 새로운 관점
- 6장: 기존 생각의 전환점
- 7장: 구체적 변화 행동의 시작
- 8장: 일상에서의 실천과 시행착오
- 9장: 현재 시점에서의 성숙한 관점
```

## 🎨 2. 장르별 특수 어체 및 문체 지침

### 문학 장르별 세부 지침

**수필 (에세이)**
- **어체**: 1인칭 관찰자 시점, "~했다", "~인 것 같다", "~하더라" 등 성찰적 어미 활용
- **문체**: 개인적 경험과 보편적 메시지의 자연스러운 연결
- **특징**: 은유와 비유를 통한 감성적 표현, 독백적 성찰의 리듬감
- **구성**: 구체적 경험 → 내적 성찰 → 보편적 깨달음의 흐름
- **인간적 요소**: 
  * 솔직한 감정 토로와 개인적 실패담
  * "사실은...", "그런데 말이다...", "생각해보니..." 등 자연스러운 화제 전환
  * 때로는 논리적 결론 없이 감정적 여운으로 마무리
  * 일상적 표현과 방언, 개인적 어투 자연스럽게 섞기

**소설**
- **어체**: 서술자 시점에 따른 일관된 문체 (1인칭/3인칭 선택)
- **문체**: 대화체와 서술체의 균형 잡힌 조화
- **특징**: 인물의 심리 묘사와 상황 전개의 생동감
- **구성**: 인물-사건-배경의 유기적 결합, 갈등과 해결의 극적 구성
- **인간적 요소**:
  * 인물의 불완전함과 모순적 행동
  * 예측 불가능한 대화와 상황 전개
  * 감정의 미묘한 변화와 내적 갈등
  * 서술자의 주관적 개입이나 편견 드러냄

**시**
- **어체**: 압축적이고 상징적 언어, 운율과 리듬 고려
- **문체**: 감정과 이미지의 절제된 표현
- **특징**: 여백과 침묵의 의미, 은유와 상징의 층위
- **구성**: 감정의 흐름에 따른 자유로운 행 배치
- **인간적 요소**:
  * 갑작스러운 감정의 폭발이나 잠잠함
  * 일상어와 시적 언어의 자연스러운 교차
  * 개인적 기억이나 상처의 생생한 형상화
  * 완벽하지 않은 운율과 자연스러운 리듬감

### 비문학 장르별 세부 지침

**자기계발서**
- **어체**: "~해보세요", "~하시기 바랍니다", "~할 수 있습니다" 등 권유형/격려형
- **문체**: 친근하면서도 확신에 찬 톤, 독자와의 대화체
- **특징**: 구체적 실행 방법과 동기 부여의 균형
- **구성**: 문제 인식 → 해결책 제시 → 실행 가이드 → 성공 사례
- **인간적 요소**:
  * 저자의 개인적 실패담과 극복 과정
  * "솔직히 말하면", "제 경험으로는" 등 주관적 표현
  * 때로는 확신에 찬 어투, 때로는 겸손한 제안
  * 독자에게 직접 말하는 듯한 친근한 어투

**인문학 도서**
- **어체**: "~이다", "~한다", "~할 것이다" 등 단정적, 학술적 문체
- **문체**: 격식 있고 논리적인 전개, 적절한 인용과 근거 제시
- **특징**: 깊이 있는 사유와 체계적 논증
- **구성**: 개념 정립 → 역사적 맥락 → 현대적 해석 → 미래 전망
- **인간적 요소**:
  * 저자의 개인적 해석과 독창적 관점
  * "필자는 생각한다", "개인적으로는" 등 주관적 판단
  * 때로는 감정적 몰입이 논리를 압도
  * 학술적 엄밀함과 개인적 경험의 자연스러운 결합

**심리학 도서**
- **어체**: "연구에 따르면", "~로 나타났다", "~할 수 있다" 등 객관적이면서 공감적
- **문체**: 과학적 근거와 인간적 이해의 조화
- **특징**: 전문성과 대중성의 균형, 사례를 통한 쉬운 설명
- **구성**: 과학적 배경 → 현상 분석 → 실제 사례 → 적용 방법
- **인간적 요소**:
  * 저자의 임상 경험이나 개인적 통찰
  * 독자에 대한 공감과 이해의 표현
  * 때로는 전문적 확신, 때로는 조심스러운 제안
  * 인간적 따뜻함이 담긴 조언과 격려

**사회/정치/경제 도서**
- **어체**: "~로 보인다", "~할 것으로 예상된다", "~해야 한다" 등 신중하면서 제안적
- **문체**: 객관적 분석과 균형 잡힌 시각
- **특징**: 데이터 기반 논증과 다각적 관점 제시
- **구성**: 현상 진단 → 원인 분석 → 해결 방안 → 정책 제언
- **인간적 요소**:
  * 저자의 정치적 성향이나 가치관 자연스럽게 드러냄
  * 현실에 대한 우려나 희망의 감정적 표현
  * 때로는 강한 주장, 때로는 조심스러운 분석
  * 개인적 경험이나 목격담의 생생한 서술

**실용서**
- **어체**: "~하세요", "~해야 합니다", "~하면 됩니다" 등 명령형/안내형
- **문체**: 명확하고 직접적인 지시, 단계별 설명
- **특징**: 즉시 적용 가능한 구체적 방법론
- **구성**: 기초 개념 → 단계별 실행 → 문제 해결 → 응용 발전
- **인간적 요소**:
  * 저자의 실패 경험과 시행착오 과정
  * "제가 해보니", "경험상" 등 개인적 증언
  * 독자의 어려움에 대한 공감과 격려
  * 때로는 단호한 지시, 때로는 부드러운 권유

## 📚 3. 장르별 매력적인 제목 작성 가이드

### 제목 작성의 5가지 핵심 원칙
1. **타겟 독자의 관심사 직접 반영**
2. **궁금증과 호기심을 유발하는 요소 포함**
3. **기억하기 쉽고 입에 잘 붙는 어감**
4. **장르적 특성과 내용의 톤 일치**
5. **차별화된 관점이나 독창성 암시**

### 문학 장르 제목 특징

**수필 (에세이)**
- **패턴**: 개인적 경험 + 보편적 메시지
- **어감**: 진솔하고 내밀한 톤
- **예시**: 
  - "서른, 혼자가 편해졌습니다" (나이+상황+감정)
  - "아버지의 마지막 선물" (관계+시간+의미)
  - "그 여름, 나는 어른이 되었다" (시간+변화+성장)

**소설**  
- **패턴**: 시간/장소 + 인물/상황 + 장르적 특성
- **어감**: 호기심을 유발하는 미스터리감
- **예시**: 
  - "1960년 부산역 세 번째 플랫폼" (시간+장소+구체성)
  - "시간을 건너온 여의사" (판타지+직업+신비감)
  - "열두 번의 이별 연습" (반복+감정+성장)

**시**
- **패턴**: 감성적 이미지 + 시적 운율
- **어감**: 함축적이고 아름다운 표현
- **예시**: 
  - "계절이 내게 말을 걸 때" (의인법+대화+시간)
  - "도시의 외로운 불빛들" (공간+감정+복수형)
  - "바람이 전하는 이야기" (자연+전달+서사)

### 비문학 장르 제목 특징

**자기계발서**
- **패턴**: 문제/목표 + 해결/방법 + 베네핏
- **어감**: 희망적이고 실행 가능한 느낌
- **예시**: 
  - "번아웃 없는 삶의 기술" (문제 해결+삶+방법론)
  - "내향인의 당당한 네트워킹" (특성+감정+기술)
  - "30분 루틴으로 바뀐 인생" (시간+방법+결과)

**인문학 도서**
- **패턴**: 주제/개념 + 새로운 관점 + 시대적 의미
- **어감**: 지적이고 깊이 있는 톤
- **예시**: 
  - "한국인의 미적 DNA" (정체성+특성+본질)
  - "디지털 시대, 관계의 재발견" (시대+주제+새로운 관점)
  - "철학이 묻고 일상이 답하다" (학문+질문+삶)

**심리학 도서**
- **패턴**: 심리적 현상 + 이해/해결 + 일상적 적용
- **어감**: 공감하면서도 희망적인 톤
- **예시**: 
  - "완벽주의라는 감옥" (문제+은유+제약)
  - "마음의 상처, 일상의 치유" (문제+대조+해결)
  - "불안과 친구가 되는 법" (감정+관계+방법)

**사회/정치/경제 도서**
- **패턴**: 사회 현상 + 분석 관점 + 미래 전망
- **어감**: 객관적이면서 통찰력 있는 톤
- **예시**: 
  - "대한민국 저출산 리포트" (국가+현상+분석)
  - "MZ세대가 말하는 일의 미래" (세대+주제+전망)
  - "부동산, 욕망의 지도" (주제+감정+공간)

**실용서**
- **패턴**: 구체적 목표 + 실행 방법 + 성공 암시
- **어감**: 확신에 차고 실용적인 톤
- **예시**: 
  - "주식투자 절대공식" (분야+방법+확신)
  - "1인 가구 살림의 정석" (대상+주제+완성도)
  - "부업으로 월 100만원 만들기" (방법+목표+구체성)
